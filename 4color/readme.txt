Coloring the map
-------------------------------------------------------------------------------------------------------
* If two countries share a common border, these countries 
  will be painted in different colors
* You can return to the previous state by button "Undo" (or ctrl + Z)
* In mathematics, the four color theorem, or the four color map theorem, 
  states that, given any separation of a plane into contiguous regions, 
  producing a figure called a map, no more than four colors are required
  to color the regions of the map
  so that no two adjacent regions have the same color.
* You can save map by Save button (or ctrl + S)
* You can make polygon by Make buttton
* You can make points in right order
* Interface automaticaly connect first and last point of polygon
* File's format:
  line: [country's number]:[neightbour's number]*
